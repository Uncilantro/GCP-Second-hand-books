# GCP-Second-hand-books
一个校园二手书查询小程序，暂时仅仅支持线下交易和每个年级需要购买的书籍查询
## 项目启动
* 1.打开终端输入`cd server`
* 2.输入`node index.js`
* 3.在uniapp上方选择微信小程序运行
